# 🤖 AI Creator Portfolio

一个视觉效果炫酷的个人 AI 作品展示网站。

## ✨ 特性

- 🎨 **炫酷视觉效果** - 霓虹渐变色 + 粒子动画
- 📱 **响应式设计** - 支持桌面和移动端
- ⚡ **流畅动画** - 滚动触发动画 + 悬停效果
- 🎯 **简洁架构** - 纯 HTML/CSS/JS，无需构建

## 🛠️ 技术栈

- HTML5
- CSS3 (动画、渐变、Flexbox/Grid)
- Vanilla JavaScript
- Canvas API (粒子效果)

## 📦 项目结构

```
ai-portfolio/
├── index.html      # 主页面
├── styles.css      # 样式文件
├── script.js       # 交互逻辑
└── README.md       # 项目说明
```

## 🚀 使用方法

1. Clone 仓库
2. 用浏览器打开 `index.html`
3. 或部署到任意静态托管服务
